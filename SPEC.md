# KeyAuth Clone - Specification

## Project Overview
- **Project name**: KeyAuth Clone
- **Type**: SaaS Landing Page
- **Core functionality**: Authentication service landing page with features, pricing, and documentation links
- **Target users**: Developers looking for software licensing/authentication solutions

## UI/UX Specification

### Layout Structure
- **Header**: Fixed navbar with logo, nav links, and auth buttons
- **Hero**: Full-width hero with headline, subtext, and CTAs
- **Features**: Grid layout showcasing key features
- **Pricing**: 3-column pricing cards
- **Sections**: Mobile app, Analytics, Blog, FAQ, Footer

### Visual Design

#### Color Palette
- **Background**: `#0a0a0f` (dark)
- **Surface**: `#12121a` (cards)
- **Primary**: `#6366f1` (indigo)
- **Primary Hover**: `#818cf8`
- **Accent**: `#22d3ee` (cyan)
- **Text Primary**: `#f8fafc`
- **Text Secondary**: `#94a3b8`
- **Border**: `#1e1e2e`

#### Typography
- **Font**: `Inter` (Google Fonts)
- **Headings**: Bold, large sizes
- **Body**: Regular, readable sizes
- **Hero H1**: 4rem / 64px
- **Section H2**: 2.5rem / 40px

#### Spacing
- Section padding: 80px vertical
- Container max-width: 1200px
- Card padding: 24px

### Components
- Navbar with transparent background
- Hero with gradient text effect
- Feature cards with icons
- Pricing cards with hover effects
- CTA buttons with glow effects
- Accordion for FAQ
- Footer with links

## Functionality Specification

### Core Features
1. Responsive navigation
2. Smooth scroll to sections
3. Interactive pricing toggle (monthly/yearly)
4. FAQ accordion
5. Mobile responsive design

### User Interactions
- Nav links scroll to sections
- Pricing toggle switches prices
- FAQ items expand/collapse
- Buttons have hover states
